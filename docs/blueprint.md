# **App Name**: Med360 Platform

## Core Features:

- Doctor Appointment System: Allow patients to book, modify, or cancel appointments with doctors.
- Medication History: Track prescribed medicines with dosage reminders for patients.
- Scan/X-ray Reports Upload: Store and access diagnostic records such as scan and X-ray reports. 
- Women's Health Tab: Provide a dedicated section for pregnancy tracking and maternal care for women's health.
- Payment History: Record consultation and medical payments for patients.
- Insurance Details Management: Manage insurance policy information for patients.
- Patient ID System: Assign a unique health ID to each patient for streamlined record management and secure data access using Azure SQL.

## Style Guidelines:

- Primary color: A calming blue (#5DADE2) to evoke trust and health.
- Background color: A light, desaturated blue (#EBF4FA) to provide a clean and calming backdrop.
- Accent color: A gentle green (#A3E4D7) for interactive elements, signifying growth and well-being.
- Headline font: 'Space Grotesk', a sans-serif font, giving a computerized and techy look.
- Body font: 'Inter', a sans-serif font for a modern and neutral look that complements headlines, but provides improved readability at small sizes.
- Use clean, modern icons to represent different features and health categories.
- Maintain a clean, intuitive layout with clear sections for each feature.
- Use subtle transitions and animations to enhance user experience when navigating between sections.
